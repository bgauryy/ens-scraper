# ENS-scraper

A POC for **ENS** (https://app.ens.domains/) scrapping tool

 **~0.45 seconds per value check**

 
> This is an expirimental code!
> Feel free to fork or contribute


 

##### Requirements
- yarn >= 3.2.0
- node >=16
##### Run 
- yarn install
- yarn start
